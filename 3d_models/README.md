# 3D Printed Case

This directory contains 3D printable files for the ESP32 SmartWatch.

## Files
- `Case.stl`: A basic enclosure for the ESP32, OLED, and battery.

## Printing Settings
- **Material**: PLA or PETG Recommended
- **Infill**: 20%
- **Supports**: Yes (depending on orientation)
- **Layer Height**: 0.2mm

*Note: This case is a starting point. Modifications might be needed depending on your specific battery size and button placement.*
